# Grade Calculator

A simple, responsive web-based tool built with HTML, CSS, and JavaScript to help students quickly calculate their academic performance.

---

## 🚀 Live Demo

Check out the live version here:  
👉 **[https://vishnu-webdev.github.io/js-grade-calculator/](https://vishnu-webdev.github.io/js-grade-calculator/)**

---

## ✨ Features

* **Clean Interface:** A responsive and straightforward user experience designed for quick calculations.
* **JavaScript-Powered:** Uses robust conditional logic to accurately compute grades and performance status.
* **Interactive Feedback:** Utilizes prompts and alerts to guide users seamlessly through the data entry process.

---

## 🛠️ Built With

* **HTML5** – Structure and semantic markup
* **CSS3** – Styling and layout design
* **JavaScript** – Core logic, calculations, and conditional evaluations
